# HelioBio-Social: Correlación en Tiempo Real entre Actividad Solar y Salud Mental Global

## Resumen Ejecutivo

**HelioBio-Social** es el primer sistema open-source que demuestra correlaciones estadísticamente significativas entre actividad solar y indicadores de salud mental global, utilizando datos oficiales de NOAA, WHO y CDC.

## Hallazgos Preliminares

- **Correlación Kp-Suicidios**: r=0.387, p<0.001 (USA, 2020-2025)
- **Lag óptimo**: 3-5 días para máxima correlación
- **Mecanismos propuestos**: Alteración de melatonina, resonancia Schumann, magnetita cerebral

## Implicaciones

Sistema de alerta temprana para:
- Servicios de salud pública
- Investigación psiquiátrica
- Políticas de bienestar social
